# PERMISSIONS - DROITS D’ACCES BASH

r
 : droit de lecture (read) =
4

w
 : droit d'écriture (write) =
2

x
 : droit d'exécution (execute) =
1

## Pour changer les

droits d’accès

```bash
sudo chmod -R 775 /folder_name
```

## Pour changer le

propriétaire/groupe

```bash
sudo chown -R proprietaire:groupe /folder_name
```
