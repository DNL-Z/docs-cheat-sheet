# HOMEBREW

**Homebrew**
 is a package manager for MacOS and **Linux**.

Formulae
 => source code that will be compiled

Casks
 => source code are pre-compiled, with some transformation in the code, and after is compiled

## Version

```bash
brew --version
```

## Packages maintenance

```bash
brew doctor
```

## Update & Upgrade

```bash
brew update
```

```bash
brew upgrade
```

## Help

```bash
brew help
```

## Informations

```bash
brew info
```

## List packages

```bash
brew list
```

## Research packages

```bash
brew search
```

## Install Cask

```bash
brew install --cask cask_name
brew install --cask --appdir /Users/DNL/Applications cask_name
```

## Uninstall Cask

```bash
brew uninstall --cask cask_name
```
