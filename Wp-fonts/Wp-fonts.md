# Ajouter des Fonts sur **WordPress**

	1.
Google Fonts ou générer sur \*HYPERLINK "https://www.fontsquirrel.com/"https://www.fontsquirrel.com/
	2.
Aller dans « Generator », choisir « Expert »
	3.
Cocher tous les différents formats
	4.
Download your kit
	5.
Ouvrir le stylesheet.css, copier le @font-face dans le fichier style.css, attention aux URL dans la font face(font/nomdelapolice)
	6.
Copier les fichiers dans le repertoire fonts
